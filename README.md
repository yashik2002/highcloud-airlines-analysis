# ✈️ HighCloud Airlines Performance Dashboard

## 📊 Project Overview

The HighCloud Airlines Performance Dashboard is a data analytics project that analyzes airline performance using passenger data, routes, load factors, and distance metrics.

This dashboard helps understand airline efficiency, passenger preferences, and route performance using interactive visualizations.

---

## 🎯 Objectives

* Calculate date fields from raw data
* Analyze Load Factor Percentage
* Identify Top Airlines
* Find Top Routes
* Compare Weekday vs Weekend Load
* Analyze Flights by Distance
* Build Interactive Dashboard

## 📈 Key Analysis

### 1. Date Analysis

Created calculated fields:

* Year
* Month Number
* Month Name
* Quarter
* Year-Month
* Weekday Name
* Financial Month
* Financial Quarter

---

### 2. Load Factor Analysis

Load Factor Formula:

Load Factor (%) = (Passengers / Seats) × 100

Analysis includes:

* Yearly Load Factor
* Quarterly Load Factor
* Monthly Load Factor
* Carrier-wise Load Factor

---

### 3. Top Airlines

Top 10 airlines identified based on:

* Passenger Count
* Airline Preference

---

### 4. Top Routes

Most popular routes based on:

* Number of Flights
* From City to Destination City

---

### 5. Weekday vs Weekend

Compared airline traffic between:

* Weekdays
* Weekends

---

### 6. Distance Analysis

Flights grouped into:

* Short Distance
* Medium Distance
* Long Distance

---

### 7. Filters

Dashboard filters include:

* Source Country
* Source State
* Source City
* Destination Country
* Destination State
* Destination City
